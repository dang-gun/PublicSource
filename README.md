## How to download a part (부분 다운로드 방법)
You can partially download source files from GitHub by using "DownGit". \
'DownGit'를 이용하여 깃허브의 소스를 부분 다운로드 할 수 있습니다.

[DownGit Link(DownGit 바로가기)](https://downgit.github.io/#/home)
